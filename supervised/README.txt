Code Running Instructions:
NOTE: Must have pyplot as well as sklearn installed on the system


1. Decision Trees
        cd decisionTree
python randomForestDigits.py (Run for Digit Dataset)
python randomForestWines.py (Run for Wine Dataset)


1. Neural Networks
        cd neural
        python wineBrain.py
        python digitBrain.py


1. Boosting
cd boosting
python boostingImages.py
python boostingWines.py


1. SVM
        cd svm
        python svm_digits.py
        python svm_wines.py


1. KNN
        cd knn
        python knn_digits.py
        python knn_wine.py
